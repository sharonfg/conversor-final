# Conversor de Divisas Profesional

Aplicación web para convertir divisas con tasas en tiempo real.

## Características
- Conversión entre 160+ monedas
- Interfaz moderna y responsive
- Tasas actualizadas cada 5 minutos
- Animaciones y efectos visuales
- Validación de formularios
- Historial de tasas populares

## Tecnologías
- HTML5, CSS3, JavaScript (ES6+)
- CurrencyFreaks API
- Azure Blob Storage
- Font Awesome icons
- Google Fonts

## Despliegue
Desplegado en Azure Blob Storage como aplicación estática.
